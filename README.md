# Dostaffkin

Приложение для доставки посылок между странами и внутри стран. Обеспечивает безопасную, быструю доставку с отслеживанием 24/7.

## Функционал

- **Главная страница**: Описание услуг и навигация
- **Заказ доставки**: Форма с выбором маршрута, размера посылки, расчетом стоимости
- **Отслеживание**: Поиск посылки по номеру с информацией о статусе и маршруте

## Технологический стек

- **Frontend**: Angular 21, TypeScript
- **Библиотеки**: RxJS, ngx-toastr
- **API**: HTTP клиент для взаимодействия с сервером доставки

## Установка и запуск

```bash
npm install
npm start
```

Приложение запустится на `http://localhost:4200`.
